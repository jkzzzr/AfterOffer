程序设计：
		池,Builder：
			在文档之间的相似度时候，采用了池的想法，用静态变量，存储所有文档之间的相似度，如果这个变量中有你想要的，就直接返回，没有的话就临时计算
			再将计算好的返回给你，同时存储进静态变量中。
		解耦和：
			将和查询无关的东西，比方说只考虑文档的东西，计算出来。将与查询有关的东西，分离出来，到时候计算。
		清理功能：
			每年的数据做完之后，将其中的某些变量清空掉，这样防止内存泄漏，这样，可以无缝运行下一年的数据。
